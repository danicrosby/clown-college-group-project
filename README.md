## Get Started

```terminal
$ git clone git@github.com:nss-evening-cohort-14/clown-college-dashboard-clown-college-dashboard.git
$ cd clown-college-dashboard-clown-college-dashboard
$ npm i
$ npm start
```

## Features

- Google Authentication
  * If unauthenticated the application is Read Only
  * If authenticated the application, the user can do full CRUD

#### Students

* View all Students
* Add a Student
* Edit a Student
* Remove a Student

#### Teachers

* View all active Teachers
* Create a Teacher
* Edit a Teacher
* Delete a Teacher

#### Lessons

* View all the Lessons
* Add a Lesson
* Edit a Lesson
* Delete a Lesson

#### Classes

* View all the Classes
* Add a Class
* Edit a Class
* Delete a Class

#### Majors

* View all the Majors
* Add a Major
* Edit a Major
* Delete a Major
